Things to remember--

Chapter 9-The Window Object

  • Every JavaScript environment has a global oject.
  • The Browser Object Model(BOM) is a collection of properties and methods that contain information about the brower and computer screen
  • Global variables are properties of the window object in a brower environment.
  • alert, confirm(), prompt() are all methods of the window object and open dialogs that halt the execution of the program.
  • window.navigator object gives info about the user's browser and operating system, although it can be unreliable.
  • window.location object provides info about the URL of current page
  • window.history object keeps info about pages that have been visited in your session.
  • You can open, close, resize and move windows--but be careful in doing so.
  • window.screen object provides info about the user's screen.
  • document.write() is and old way of writing text to the document and shouldn't be used.
  • Cookies can be used to store small pieces of info between requests using the document.cookie property.
  • window.setTimeout() method can be used to invoke a function after a swet amount of time. Cancel it using the clearTimeout() method.
  • window.setInterval() method can be used to repeatedly invoke a function. Stop it using clearInterval() methodl.
  • window.requestAnimationFrame() method can be sued to produce smooth and optimized animation by using the browser's built-in graphics capabilities. Cancel it using cancelAnimationFrame() method.

    • Chapter 14-HTML5 APIs

  • HTML5.1 is the latest of the Hypertext Markup Language. Covers a variety of technologies including several APIs that are accessible using JavaScript
  • data- attributes help to embed custom data into a web page that can then be used to enhance the user experience with JavaScript.
  • The Web Storage API allows key-value pairs to be stored on the user's device similar to cookies but without the same storage restrictions.
  • Geolocation API allows you to access the geographic coordinates of the user's device, as long as the user gives permission.
  • Web Worker API can be used to perform computationally intensive tasks in the background, which helps to avoid websites becoming unresponsive.
  • Websockets are a new protocal for communicating over the Internet, and allow real-time, two-way communcation.
  • Notification API allows you to display notifications on the user's system.
  • canvas element can be used to dynamically draw geometric shapes, text, and images on a web page in real-time using JavaScript.
  • shim or polyfill is a piece of code that adds support of missing features to older browsers.
